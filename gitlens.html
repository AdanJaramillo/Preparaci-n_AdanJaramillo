GitLens
Funcionalidades

• Navegación de revisión sin esfuerzo (hacia atrás y hacia adelante) a través del historial de un archivo
• Una anotación discreta de culpa de la línea actual al final de la línea que muestra la confirmación y el autor que modificó la línea por última vez, con información de culpa más detallada accesible en hover
• Lente de código de autoría que muestra la confirmación más reciente y el número de autores en la parte superior de los archivos y / o en los bloques de código.
• Una anotación de culpa en la barra de estado que muestra la confirmación y el autor que modificó por última vez la línea actual
• Anotaciones de archivos bajo demanda en el margen del editor, que incluyen o culpa : muestra la confirmación y el autor que modificó por última vez cada línea de un archivo o cambios : resalta los cambios locales (no publicados) o las líneas modificadas por la confirmación más reciente o mapa de calor : muestra qué tan recientemente se cambiaron las líneas, en relación con todos los demás cambios en el archivo y hasta ahora (caliente frente a frío)
• Muchas vistas ricas del side bar o Un commits ver a visualizar, explorar y administrar confirmaciones git
o Una vista de repositorios para visualizar, explorar y administrar repositorios de git
o Una vista de historial de archivos para visualizar, navegar y explorar el historial de revisión del archivo actual o solo las líneas seleccionadas del archivo actual
o Una vista de historial de líneas para visualizar, navegar y explorar el historial de revisión de las líneas seleccionadas del archivo actual
o Una vista de ramas para visualizar, explorar y administrar las ramas de git
o Una vista remotos para visualizar, explorar y administrar controles remotos y sucursales remotas de git
o Una vista de stashes para visualizar, explorar y administrar los escondites de git
o Una vista de etiquetas para visualizar, explorar y administrar etiquetas de git
o Una vista colaboradores para visualizar, navegar y explorar colaboradores
o Una vista de búsqueda y comparación para buscar y explorar historiales de confirmaciones por mensaje, autor, archivos, identificación, etc., o visualizar comparaciones entre ramas, etiquetas, confirmaciones y más
• Una paleta de comandos de git para proporcionar acceso guiado (paso a paso) a muchos comandos de git comunes, así como un acceso rápido a
o confirma - historial y búsqueda
o escondites
o estado : estado actual de la rama y del árbol de trabajo
• Un editor de rebase interactivo fácil de usar para configurar fácilmente una sesión de rebase interactiva
• enlaces de terminal :ctrl+clicken enlaces automáticos en el terminal integrado para saltar rápidamente a más detalles para confirmaciones, ramas, etiquetas y más
• integraciones de proveedores remotos enriquecidos : github, gitlab, gitea, bitbucket, azure devops
o Emitir y extraer el enlace automático de la solicitud
o Información de desplazamiento enriquecida proporcionada para problemas vinculados y solicitudes de extracción (solo github)
o Asocia solicitudes de extracción con ramas y confirmaciones (solo github)
• Muchos comandos poderosos para navegar y comparar revisiones y más
• Personalizables menús y barras de herramientas de control sobre dónde se muestran los elementos de menú y barra de herramientas
• modos definidos por el usuario para alternar rápidamente entre conjuntos de configuraciones
